About the Algorithms

Breadth-First Search (BFS)

BFS is an uninformed search algorithm that explores all nodes at the current depth level before moving to the next level. It uses a queue (FIFO) for traversal and guarantees the shortest path in an unweighted graph.

Time Complexity: O(V + E)
Space Complexity: O(V)

Depth-First Search (DFS)

DFS is another uninformed search algorithm that explores as far as possible along a branch before backtracking. It uses a stack (LIFO) or recursion for traversal. DFS does not guarantee the shortest path but can explore deep structures with less memory.

Time Complexity: O(V + E)
Space Complexity: O(V)

A* Search

A* Search is an informed search algorithm that uses both the cost to reach a node from the start (g(n)) and a heuristic estimate of the cost to the goal (h(n)) to decide the next node to explore. It is optimal when the heuristic is admissible (does not overestimate the true cost).

Time Complexity: O(V + E)
Space Complexity: O(V)

How to Use the Visualizer

The Navigation Bar

The “Algorithms” menu allows you to choose between three algorithms: Depth-First Search (DFS), Breadth-First Search (BFS), and A* Search. Hover over the menu and select the algorithm you want to visualize.

Add Walls

Use the “Add Walls” option to create obstacles on the grid. Hold down the mouse button and drag to place walls. Release the button when you’re done.

Remove Walls

Use the “Remove Walls” option to clear obstacles. Hold down the mouse button and drag over walls you want to remove.

Start Visualization

The “Start Visualization” button begins the algorithm. Make sure you’ve selected an algorithm, start, and finish points before starting.

Select Start and Finish

The “Select Start” and “Select Finish” options let you choose the starting and ending cells. Click on a cell to mark it as the start or finish.

Clear Board

The “Clear Board” button resets the grid, removing all walls, start, and finish points. Use this before re-running an algorithm.
